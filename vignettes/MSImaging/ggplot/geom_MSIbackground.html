geom_MSIbackground {ggplot} R Documentation

config of the background of the MS-imaging charting plot.

Description

Usage

geom_MSIbackground(background);

Arguments

background

the background color value or character vector TIC or BPC.

Details

Authors

BioNovoGene

Value

this function returns clr object in types based on the `background` parameter: 1. "TIC" or "BPC": MSITICOverlap 2. html color code, or supported gdi image its file path: MSIBackgroundOption

clr value class

  • MSIBackgroundOption
  • MSITICOverlap
