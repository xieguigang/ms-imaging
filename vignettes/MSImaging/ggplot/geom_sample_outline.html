geom_sample_outline {ggplot} R Documentation

Create a plot layer of outline for the sample data

Description

Usage

geom_sample_outline(
    spots = NULL,
    threshold = 0,
    scale = 5,
    degree = 20,
    resolution = 1000,
    q = 0.1,
    line.stroke = "stroke: white; stroke-width: 6px; stroke-dash: solid;");

Arguments

threshold

the intensity threshold for make spatial spot binariation, for clean sample, leaves this parameter default zero, for sample with background, set this parameter in range (0,1) for make spot cutoff. [as double]

scale

contour tracing pixel rectangle scale size. [as integer]

degree

[as double]

resolution

[as integer]

q

[as double]

line.stroke

a lineElement that create via the ggplot function: element_line.
