raster_blending {ggplot} R Documentation

rendering a gdi+ heatmap for create raster annotation in ggplot layer

Description

Usage

raster_blending(pixels, dims,
    scale = "gray",
    levels = 255,
    filters = NULL,
    backcolor = "black");

Arguments

pixels

the pixels data

dims

the spatial dimension size of the sample data

scale

the color palette name. [as string]

levels

the color scaler levels. [as integer]

filters

the raster filter object, data type could be RasterPipeline or the ggplot wrapper of the pipeline: MSIFilterPipelineOption.

env

[as Environment]
