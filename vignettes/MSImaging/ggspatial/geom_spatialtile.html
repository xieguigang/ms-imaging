geom_spatialtile {ggspatial} R Documentation

add a spatial overlaps of the STdata on current SMdata imaging

Description

Usage

geom_spatialtile(tile,
    geneID = NULL,
    STdata = NULL,
    colorSet = "grays",
    size = 13);

Arguments

tile

A collection of the spatial spot mapping. [as SpatialMapping]

STdata

the spatial transcriptomics data matrix

geneID

the gene ID for pull the spatial expression from the `STdata`. [as string]

colorSet

the color set data for rendering the gene expression spatial heatmap

size

the spot size when drawing the spatial expression. [as double]
