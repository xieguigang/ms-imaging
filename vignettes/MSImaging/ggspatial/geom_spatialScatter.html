geom_spatialScatter {ggspatial} R Documentation

draw scatter layer with a given x and y coordinates.

Description

Usage

geom_spatialScatter(x, y, colors,
    size = 3,
    alpha = 0.8);

Arguments

x

a numeric vector of pixel x

y

a numeric vector of pixel y

colors

the colors for rendering each scatter spot

size

the spot size for the drawing

alpha

the transparency alpha value for drawing the scatter points.

env

-

Details

Value

this function returns data object of type SpatialAnnotationScatterLayer.

clr value class

  • SpatialAnnotationScatterLayer
