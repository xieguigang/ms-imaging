.auto_size_internal {MSImaging} R Documentation

Internal canvas dimension calculator

Description

Helper function that computes base canvas dimensions with padding and scaling.

Usage

.auto_size_internal(dims, padding,
    scale = 1);

Arguments

dims

Original image dimensions (numeric vector [width, height])

padding

Numeric vector specifying padding [top, right, bottom, left]

scale

Numeric scaling factor for initial dimension adjustment

Details

Authors

xieguigang

Value

Numeric vector [width, height] representing: (original dimensions * scale) + padding adjustments
