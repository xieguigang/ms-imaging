MSI_sampleTIC {MSImaging} R Documentation

Create raster image based on the total ions thumbnail of the sample rawdata file.

Description

Usage

MSI_sampleTIC(rawdata,
    dims = NULL,
    filters = NULL);

Arguments

rawdata

should be the mzkit mzpack data object

filters

the raster filters for the ms-imaging spatial pixels rawdata

Details

Authors

xieguigang

Value

a gdi+ raster image data object for make single ion ms-imaging rendering as background.
