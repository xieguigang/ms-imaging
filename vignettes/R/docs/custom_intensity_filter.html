custom_intensity_filter {MSImaging} R Documentation

Apply Custom Filter Sequence to MSI Ion Layer

Description

Processes an ion layer using user-defined raster filters followed by automatic intensity scaling through Trimmed Quantile Scaling (TrIQ).

Usage

custom_intensity_filter(ion, filters,
    MSI.TrIQ = 0.8);

Arguments

ion

An `ion` object containing MSI data with `layer` raster data

filters

List of raster filters to apply in sequence (function objects)

MSI.TrIQ

Quantile threshold for intensity truncation (default 0.8)

Details

Key features: - Allows custom filter sequences via \code{applyrasterfilter()} - Maintains automatic TrIQ scaling after custom filtering - Preserves original object structure while updating `layer` and `TrIQ`

Authors

xieguigang

Value

Modified `ion` object with filtered `layer` and updated `TrIQ` values
