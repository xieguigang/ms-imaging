geom_blanket {MSImaging} R Documentation

Create monochromatic overlay for MS imaging

Description

Generates a grayscale ggplot2 layer for mass spectrometry imaging (MSI) data, typically used as a background layer or for intensity normalization.

Usage

geom_blanket(mz,
    tolerance = 'getOption'('mzdiff', 'default' = 'da:0.3'));

Arguments

mz

numeric. The target mass-to-charge ratio (m/z) value for ion detection.

tolerance

character or numeric. Mass accuracy tolerance. See geom_red() for format details.

Details

Authors

xieguigang

Value

A ggplot2 layer object using "Greys" color palette with alpha channel (c8 = 200/255 transparency) suitable for creating intensity-normalized backgrounds or composite overlays.

Examples

#\dontrun{

plot_base() + geom_blanket(1200.834)

#}
