geom_green {MSImaging} R Documentation

Create green channel MS imaging layer

Description

Usage

geom_green(mz,
    tolerance = 'getOption'('mzdiff', 'default' = 'da:0.3'));

Arguments

mz

numeric. The target mass-to-charge ratio (m/z) value for ion detection.

tolerance

character or numeric. Mass accuracy tolerance. See geom_red() for format details.

Details

Authors

xieguigang

Value

A ggplot2 layer object displaying ion distribution in green color scale.

Examples

\dontrun{ plot_base() + geom_green(556.276) }
