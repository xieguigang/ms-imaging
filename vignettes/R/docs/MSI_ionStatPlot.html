MSI_ionStatPlot {MSImaging} R Documentation

Combined Statistical Plot and Mass Spectrometry Imaging Visualization Generates a composite visualization combining statistical charts (e.g., bar/box/violin plots) with mass spectrometry imaging (MSI) data. Outputs to specified PNG file with customizable layout.

Description

Usage

MSI_ionStatPlot(mzpack, mz, met, sampleinfo,
    savePng = './Rplot.png',
    ionName = NULL,
    size = [1000],
    colorMap = NULL,
    MSI.colorset = 'viridis:turbo',
    ggStatPlot = NULL,
    padding.top = 150,
    padding.right = 200,
    padding.bottom = 150,
    padding.left = 150,
    interval = 50,
    combine.layout = [5],
    jitter.size = 4,
    TrIQ = 0.65,
    backcolor = 'black',
    regions = NULL,
    swap = FALSE,
    title.fontsize = 40,
    font.family = 'Times New Roman',
    show.legend = TRUE,
    show.grid = TRUE,
    show.stats = TRUE,
    show.axis.msi = TRUE,
    show.title = TRUE,
    tic.outline = NULL,
    intensity.format = 'G2');

Arguments

mzpack

An mzpack object containing MS imaging data and layer information. Must be compatible with ggplot MS imaging functions.

mz

Target ion mass-to-charge ratio (m/z) value for imaging.

met

List containing sample intensity data. List keys should be sample IDs with corresponding intensity values.

sampleinfo

Metadata list defining sample groups and visualization properties. Requires: \itemize{ \item \code{group}: Character vector of group names (plot titles) \item \code{id}: Character vector of sample IDs corresponding to \code{met} keys \item \code{color}: HTML color codes for each group's statistical plot elements }

savePng

Output file path for PNG image. Default: "./Rplot.png".

ionName

Display title for the ion. Uses m/z value when \code{NULL} (default).

size

Image dimensions in pixels [width, height]. Default: \code{c(2400, 1000)}.

colorMap

Named list mapping group names to custom colors (overrides \code{sampleinfo$color}).

MSI.colorset

Color palette for MS imaging. Default: "viridis:turbo".

ggStatPlot

Statistical plot geometry function. Default: bar plot (\code{geom_barplot}).

padding.top

Top padding in pixels. Default: 150.

padding.right

Right padding in pixels. Default: 200.

padding.bottom

Bottom padding in pixels. Default: 150.

padding.left

Left padding in pixels. Default: 150.

interval

Spacing between statistical plot and MS image in pixels. Default: 50.

combine.layout

Layout ratio [statplotwidth, msi_width]. Default: \code{c(4, 5)} (converted to 44.4% vs 55.6% of available width).

jitter.size

Point size for jittered data points. Default: 8.

TrIQ

Intensity quantile threshold (0-1) for MS image enhancement. Default: 0.65.

backcolor

Background color for MS imaging panel. Default: "black".

regions

Optional list with spatial coordinates to overlay on MS image: \itemize{ \item \code{x}: X-coordinates vector \item \code{y}: Y-coordinates vector \item \code{colors}: Corresponding colors vector }

Details

The function performs three main operations: 1. Computes ANOVA group statistics using \code{ANOVAGroup()} 2. Creates a statistical plot (bar/box/violin) with significance annotations 3. Generates MS ion image with optional spatial region overlays Plots are combined using a customizable split-layout system. Statistical tests include: - ANOVA global test (displayed in plot) - Pairwise t-tests against global mean (p-value symbols)
