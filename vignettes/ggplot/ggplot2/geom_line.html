geom_line {ggplot2} R Documentation

Connect observations

Description


geompath() connects the observations in the order in which they appear in
the data. geomline() connects them in order of the variable on the x axis.
geom_step() creates a stairstep plot, highlighting exactly when changes
occur. The group aesthetic determines which cases are connected together.

Usage

geom_line(
    mapping = NULL,
    color = NULL,
    width = 5,
    show.legend = TRUE,
    alpha = 1);

Arguments

Details

An alternative parameterisation is geom_segment(), where each line corresponds to a single case which provides the start and end coordinates.
